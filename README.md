# 6-lack

## 6lack is a clone based of off [Slack](https://slack.com/). It is essentially a chat room for *musicians*, designed for *artists* to share their ideas and music. 

Impelmented Technologies:
* HTML
* CSS
* JavaScript
* Postgres
* Ruby on Rails
* Webpack
* React
* Redux
* Adobe Photoshop
* Adobe Illustrator
